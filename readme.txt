== Description == 
Display word count of each post within the Edit Posts table in WP-Admin

Brought to you by Best Credit Cards, an Australian <a href="http://www.bestcreditcards.com.au/">credit card comparison</a> website.

== Installation ==
1. Upload `wordcount` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
